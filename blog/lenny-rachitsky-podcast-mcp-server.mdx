## Deploy Your Own MCP Server for Content Archives

You can deploy your own instance of the Lenny Rachitsky Podcast Transcripts MCP Server or any other content archive using the [HAPI CLI](https://docs.mcp.com.ai/components/hapi-server/hapi-cli), [Docker](https://docs.mcp.com.ai/deployment/docker), to the cloud provider of your choice, for now Cloudflare Workers and Fly.io are supported.

```bash
hapi serve lenny-rachitsky-podcast --port 3030 --headless --url https://lenny-rachitsky.search.mcp.com.ai --openapi https://docs.mcp.com.ai/servers-apis/openapi/podcast-transcripts.yaml
```

Find the MCP Server in the [MCP Registry](https://registry.modelcontextprotocol.io/?q=lenny-rachitsky) and connect it to your favorite AI tools and language models.
